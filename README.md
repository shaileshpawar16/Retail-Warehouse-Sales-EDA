Retail & Warehouse Sales EDA Project

Project Overview:

This project focuses on Exploratory Data Analysis (EDA) of a large retail and warehouse sales dataset containing more than 300,000 records. The dataset includes supplier information, item descriptions, item categories, retail sales, warehouse sales, and retail transfer values across multiple years. The objective of this project was to understand operational sales behavior, identify business patterns, analyze supplier and category performance, and create an interactive dashboard using Excel.

What We Did:

•	Explored and understood the raw dataset structure.

•	Converted raw data into structured Excel tables.

•	Investigated duplicates and validated business transaction patterns.

•	Handled missing values by replacing blank supplier values with 'NA'.

•	Analyzed negative warehouse sales and identified them as credit/refund transactions.

•	Performed supplier-wise and category-wise sales analysis.

•	Compared warehouse sales and retail sales behavior.

•	Created Pivot Tables, Pivot Charts, and KPI metrics.

•	Built an interactive dashboard using slicers and charts.

Key Insights:

•	Warehouse sales and retail sales did not always move together, showing operational differences in inventory flow.

•	Negative warehouse values were mostly related to beer and wine credits rather than data errors.

•	Some categories showed strong warehouse movement but weaker retail performance.

•	Supplier contribution varied significantly across the dataset.

•	Sales trends changed across months and years, revealing seasonal behavior.

Project Story & Analysis:

This project demonstrates how operational sales data can be transformed into meaningful business insights. The dataset tells a story about inventory movement, supplier performance, category-level behavior, and retail demand. Instead of simply creating charts, the analysis focused on understanding the business logic behind the data. The project identified operational patterns such as product credits, sales inconsistencies, supplier contribution, and category performance trends.

Dashboard Summary:

The dashboard includes KPI cards, sales trend analysis, supplier analysis, category comparison charts, consistency metrics, and interactive slicers for filtering by year and item type. It was designed to provide a business-focused overview of retail and warehouse operations.
